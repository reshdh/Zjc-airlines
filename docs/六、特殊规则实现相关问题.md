## 1. 如何实现"12小时内退票收1%手续费"的规则？

### 1.1 规则定义

**退票手续费规则：**
- **起飞前12小时内退票：** 收取订单金额的 **1%** 作为退票手续费
- **起飞前12小时外退票：** 不收取退票手续费，全额退款

### 1.2 实现逻辑

**常量定义：**
```java
private static final int URGENT_WINDOW_HOURS = 12;
private static final BigDecimal URGENT_REFUND_FEE_RATE = new BigDecimal("0.01"); // 1%
```

**判断是否在紧急窗口：**
```java
private boolean isWithinUrgentWindow(LocalDateTime departureTime) {
    if (departureTime == null) {
        return false;
    }
    LocalDateTime now = LocalDateTime.now();
    if (!departureTime.isAfter(now)) {
        return false; // 航班已起飞，不能退票
    }
    // 判断是否在起飞前12小时内
    return !departureTime.isAfter(now.plusHours(URGENT_WINDOW_HOURS));
}
```

**计算退款金额：**
```java
private BigDecimal calculateRefundAmount(Booking booking) {
    BigDecimal totalAmount = booking.getTotalAmount();
    LocalDateTime now = LocalDateTime.now();
    LocalDateTime departureTime = booking.getFlight().getDepartureTime();
    
    // 判断是否在起飞前12小时内
    boolean within12Hours = isWithinUrgentWindow(departureTime);
    
    if (within12Hours) {
        // 起飞前12小时内退票，扣除1%手续费
        BigDecimal refundFee = totalAmount.multiply(URGENT_REFUND_FEE_RATE);
        booking.setRefundFeeRate(URGENT_REFUND_FEE_RATE);
        booking.setRefundFeeAmount(refundFee);
        // 退款金额 = 订单金额 - 退票手续费
        return totalAmount.subtract(refundFee);
    } else {
        // 起飞前12小时外退票，全额退款
        booking.setRefundFeeRate(BigDecimal.ZERO);
        booking.setRefundFeeAmount(BigDecimal.ZERO);
        return totalAmount;
    }
}
```

**退票流程：**
```java
public Booking requestRefund(Long bookingId, Long userId, String reason) {
    Booking booking = getOwnedBooking(bookingId, userId);
    
    // 验证订单状态（只有已支付订单可以退票）
    if (!"PAID".equalsIgnoreCase(booking.getStatus())) {
        throw new RuntimeException("只有已出票的订单才能申请退票");
    }
    
    // 判断是否需要审核（起飞前12小时内需要审核）
    boolean within12Hours = isWithinUrgentWindow(booking.getFlight().getDepartureTime());
    boolean needsReview = within12Hours || exceedCancelLimit;
    
    if (needsReview) {
        // 需要审核
        booking.setStatus("REFUND_REVIEW");
        return bookingRepository.save(booking);
    } else {
        // 自动退款
        return finalizeRefund(booking);
    }
}
```

## 2. 如何实现"航班出发前12小时内票价上涨10%"的规则？

### 2.1 规则定义

**紧急加价规则：**
- **起飞前12小时内订票：** 在原价基础上加收 **10%** 作为紧急加价
- **起飞前12小时外订票：** 不收取紧急加价，按原价计算

### 2.2 实现逻辑

**常量定义：**
```java
private static final int URGENT_WINDOW_HOURS = 12;
private static final BigDecimal URGENT_SURCHARGE_RATE = new BigDecimal("0.10"); // 10%
```

**判断是否在紧急窗口：**
```java
private boolean isWithinUrgentWindow(LocalDateTime departureTime) {
    if (departureTime == null) {
        return false;
    }
    LocalDateTime now = LocalDateTime.now();
    if (!departureTime.isAfter(now)) {
        return false; // 航班已起飞，不能订票
    }
    // 判断是否在起飞前12小时内
    return !departureTime.isAfter(now.plusHours(URGENT_WINDOW_HOURS));
}
```

**计算订单金额：**
```java
public Booking createBooking(BookingRequest request) {
    Flight flight = flightRepository.findById(request.getFlightId())
        .orElseThrow(() -> new RuntimeException("航班不存在"));
    
    // 获取舱位价格
    BigDecimal cabinPrice = getCabinPrice(flight, request.getCabinClass());
    
    // 计算基础价格
    BigDecimal baseAmount = cabinPrice.multiply(BigDecimal.valueOf(request.getTicketCount()));
    
    // 判断是否在紧急窗口（起飞前12小时内）
    boolean urgentDeparture = isWithinUrgentWindow(flight.getDepartureTime());
    
    // 计算紧急加价
    BigDecimal surchargeRate = urgentDeparture ? URGENT_SURCHARGE_RATE : BigDecimal.ZERO;
    BigDecimal surchargeAmount = baseAmount.multiply(surchargeRate);
    
    // 计算最终订单金额
    BigDecimal totalAmount = baseAmount.add(surchargeAmount)
        .setScale(2, RoundingMode.HALF_UP);
    
    // 保存订单（包含紧急加价信息）
    Booking booking = new Booking();
    booking.setTotalAmount(totalAmount);
    booking.setUrgentSurchargeRate(surchargeRate);
    // ...
    
    return bookingRepository.save(booking);
}
```

**前端展示：**
```typescript

  <Alert status="warning">
```

### 2.3 价格计算公式

```
基础价格 = 舱位价格 × 票数
紧急加价 = 基础价格 × 紧急加价率（如果起飞前12小时内，则为10%，否则为0）
最终订单金额 = 基础价格 + 紧急加价
```

## 3. 如何实现"订票时可以添加多人信息进行订票"的功能？

### 3.1 功能概述

**多人订票规则：**
- 支持一次预订为多人购票（最多9人）
- 每位乘客需要填写：姓名、身份证号、联系电话
- 姓名可以相同（支持重名），但身份证号和联系电话必须唯一

**后端验证：**
```java
private void validatePassengerIdentities(BookingRequest request) {
    if (request == null || request.getPassengers() == null || request.getPassengers().isEmpty()) {
        return;
    }
    
    Set<String> idNumbers = new HashSet<>();
    Set<String> phones = new HashSet<>();
    int index = 0;
    
    for (BookingRequest.PassengerInfo passenger : request.getPassengers()) {
        index++;
        if (passenger == null) {
            continue;
        }
        
        String idNumber = passenger.getIdNumber() != null 
            ? passenger.getIdNumber().trim().toUpperCase() : "";
        String phone = passenger.getPhone() != null 
            ? passenger.getPhone().trim() : "";
        
        // 验证身份证号不能为空
        if (idNumber.isEmpty()) {
            throw new RuntimeException("乘客 " + index + " 的身份证号不能为空");
        }
        
        // 验证联系电话不能为空
        if (phone.isEmpty()) {
            throw new RuntimeException("乘客 " + index + " 的联系电话不能为空");
        }
        
        // 验证身份证号唯一性
        if (!idNumbers.add(idNumber)) {
            throw new RuntimeException("身份证号 " + idNumber + " 已被重复使用，请为每位乘客填写不同的证件信息");
        }
        
        // 验证联系电话唯一性
        if (!phones.add(phone)) {
            throw new RuntimeException("联系电话 " + phone + " 已被重复使用，请为每位乘客填写不同的联系电话");
        }
    }
}
```

### 4.2 时间计算

**计算时间差：**
```java
private boolean isWithinUrgentWindow(LocalDateTime departureTime) {
    LocalDateTime now = LocalDateTime.now(); // 当前时间（系统时区）
    
    // 计算时间差
    Duration duration = Duration.between(now, departureTime);
    long hours = duration.toHours();
    
    // 判断是否在12小时内
    return hours > 0 && hours <= URGENT_WINDOW_HOURS;
}
```
