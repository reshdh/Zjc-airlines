## 1. 如何优化数据库查询性能？

### 1.1 索引优化

**创建合适的索引：**
```sql
-- 为常用查询字段创建索引
CREATE INDEX idx_origin ON flights(origin);
CREATE INDEX idx_destination ON flights(destination);
CREATE INDEX idx_departure_time ON flights(departure_time);

-- 联合索引优化多条件查询
CREATE INDEX idx_route ON flights(origin, destination);
```

**索引使用原则：**
- 为 WHERE 条件字段创建索引
- 为 ORDER BY 字段创建索引
- 为 JOIN 字段创建索引
- 避免过度索引（影响写入性能）

### 1.2 查询优化

**避免全表扫描：**
```java
// ❌ 错误：全表扫描
List<Flight> flights = flightRepository.findAll();

// ✅ 正确：使用索引查询
List<Flight> flights = flightRepository.findByOriginAndDestination(origin, destination);
```

**分页查询：**
```java
Pageable pageable = PageRequest.of(page, size);
Page<Flight> flights = flightRepository.findAll(pageable);
```

**延迟加载：**
```java
@Entity
public class Flight {
    @OneToMany(fetch = FetchType.LAZY) // 延迟加载
    private List<FlightSeat> seats;
}
```

### 1.3 批量操作

**批量插入：**
```java
List<Seat> seats = new ArrayList<>();
// ... 添加座位
seatRepository.saveAll(seats); // 批量插入
```

**批量更新：**
```java
@Query("UPDATE FlightSeat fs SET fs.remainingSeats = fs.remainingSeats - :count WHERE fs.id = :id")
void updateRemainingSeats(@Param("id") Long id, @Param("count") Integer count);
```

public Booking createBooking(BookingRequest request) {
    // 保证事务隔离性

### 2.3 分布式锁（Redis，未来）

```java
public boolean tryLock(String key, int expireSeconds) {
    String value = UUID.randomUUID().toString();
    Boolean result = redisTemplate.opsForValue()
        .setIfAbsent(key, value, expireSeconds, TimeUnit.SECONDS);
    return Boolean.TRUE.equals(result);
}
```

## 3. 如何实现缓存机制？

### 3.1 Hibernate 二级缓存（可选）

**配置：**
```properties
spring.jpa.properties.hibernate.cache.use_second_level_cache=true
spring.jpa.properties.hibernate.cache.region.factory_class=org.hibernate.cache.jcache.JCacheRegionFactory
```

### 3.2 Redis 缓存（未来）

**配置：**
```properties
spring.redis.host=localhost
spring.redis.port=6379
```

**使用：**
```java
@Cacheable("flights")
public Flight getFlightById(Long id) {
    return flightRepository.findById(id).orElse(null);
}
```

### 3.3 前端缓存

**LocalStorage：**
```typescript
// 缓存用户信息
localStorage.setItem("user", JSON.stringify(user));

// 缓存航班列表
localStorage.setItem("flights", JSON.stringify(flights));
```
